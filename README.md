# Management Inventory Pro

A modern, scalable, and responsive Inventory Management System built with Flutter, Supabase, and PostgreSQL.

## Overview

Management Inventory Pro is a cross-platform inventory management solution designed for small and medium-sized businesses. The system aims to streamline inventory operations, warehouse management, purchasing, sales tracking, and reporting through a modern and responsive user experience.

Built using Flutter and Supabase, the project follows Clean Architecture principles and production-grade development practices to ensure maintainability, scalability, and long-term growth.
